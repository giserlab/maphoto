Maphoto—在地图上展示照片。

🎉 特点

- 软件仅包含一个可执行文件，无其他依赖
- 支持上传图片到服务器/腾讯云COS
- 支持多用户
- 支持地图配置

## 使用

前往[Releases][]下载对应系统的可执行二进制文件，接着运行：

```bash
maphoto serve
```

默认用户名：admin		默认密码：admin1234
